Welcome to the CAT (Copilot Adoption Team) Bootcamp Feedback Collection System. This platform enables real-time feedback collection from bootcamp participants and provides analytics for instructors and administrators.

Submit Feedback

Admin Panel

Manage events, view feedback submissions, generate QR codes, and access analytics.

How It Works

For Participants: Scan the QR code or use the unique URL provided by your instructor to submit feedback after each session.

For Instructors: Display the live count during your session to encourage participation. Access the admin panel to view detailed feedback.

For Administrators: Create events, generate QR codes, manage feedback, and view comprehensive analytics.
